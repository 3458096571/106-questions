# 想了解你的106个问题 💕

一个充满爱意的小游戏，让你通过106个问题更好地了解彼此。

## ✨ 特性

- 🎨 精美的粒子动画背景
- 💫 开场问题快闪动画
- 📝 毛玻璃效果的优雅输入框
- 📊 实时进度显示
- 💾 答案自动保存
- 📱 支持分享、保存图片和导出PDF
- 🔄 可重新开始

## 🚀 在线体验

访问 [https://3458096571.github.io/106-questions/](https://3458096571.github.io/106-questions/) 开始游戏

## 🛠️ 本地开发

```bash
# 克隆项目
git clone https://github.com/3458096571/106-questions.git

# 进入目录
cd 106-questions

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build
```
